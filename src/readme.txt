Latest stable version: 3.2

The latest stable version includes minor fixes and exports the models using the PLY-variant 
format used by 3DGSS. Using our output as the input to 3DGSS the models are automatically 
aligned and merged.

3DGSS is a command-line tool for automatic alignment and merging of range images. More 
information can be found here: cs.drexel.edu/~kon/3DGSS/
